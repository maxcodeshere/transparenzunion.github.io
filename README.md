# Transparenzunion

Prerequisites:

Ruby >= 2.5 installed

How to get started:

```
$ bundle
$ jekyll serve
```

Point your browser at http://localhost:4000

## Add a new entry

Just create a new Markdown file in the `_posts` directory. Name it accordingly to the scheme `<yyyy>-<mm>-<dd>-<title>.markdown`.
See existing posts for examples.
